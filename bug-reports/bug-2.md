# Bug 2: Employee Details Not Validated

## Title
Employee form accepts invalid data

## Severity
Medium

## Steps to Reproduce
1. Login to HRMS
2. Open Employee section
3. Enter invalid employee details
4. Click save

## Expected Result
System should validate fields and show error message.

## Actual Result
Invalid data is accepted.

## Impact
Incorrect employee information can affect payroll processing.
